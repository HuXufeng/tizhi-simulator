# 卡牌构筑式三消系统设计

## 概述
将现有的"3步顺序选任务"机制改为**杀戮尖塔风格的卡牌手牌系统**。每月发一手牌，玩家从中选3张逐一打出（上/中/下旬），打出后消耗。若3张同类型则触发三消奖励。手牌随进度增长，卡牌可被技能树改造，叙述者影响卡组构成。

## 核心循环

```
每月初 → 发手牌(5~7张) → 打第1张(上旬)
                        → 打第2张(中旬)
                        → 打第3张(下旬)
                        → 结算(同类型→三消奖励)
                        → 下个月 → 新手牌
```

## 手牌系统

### 发牌数量
| 月份 | 手牌数 |
|---|---|
| 1~6月 | 5张 |
| 7~12月 | 6张 |
| 13~24月 | 7张 |
| 25月+ | 8张 |

### 卡牌数据结构
```js
{
    id: string,
    type: 'document'|'meeting'|'research'|'report'|'inspection'|'speech'|'training'|'social'|'rest'|'study'|'event'|'curse',
    name: string,
    icon: string,
    description: string,
    energyCost: number,  // 正数消耗，负数恢复
    stressGain: number,  // 正数增加，负数减少
    rewards: { exp?: number, reputation?: number, connections?: number },
    activityType: 'work'|'social'|'rest'|'study'|'special',
    specialEffect?: string,  // 特殊卡牌效果描述
    isEvent?: boolean,
    isCurse?: boolean
}
```

## 三消奖励结算

每月3张牌全部打完后结算。若3张的 `activityType` 相同则触发三消。

| 配对 | 奖励 |
|---|---|
| 工作×3 | 声望+10, 该月所有工作类经验×1.5 |
| 社交×3 | 人脉+8 |
| 休息×3 | 精力+20, 压力-15 |
| 学习×3 | 经验+30 |
| 特殊×3 | 额外随机奖励 |

## 进度成长

### 月份解锁
- 新卡牌类型逐步解锁
- 手牌数量逐步增加
- 高月限时任务奖励更高（12个月后×1.3）

### 技能树改造
- 技能点可用于"强化"特定类型卡牌
- 强化后同一类型所有卡牌获得加成（如消耗减少/奖励增加）
- 例：花3技能点解锁「公文精通」→ 所有工作类卡精力消耗-3

### 叙述者影响
- 不同叙述者影响卡牌池权重
- 严厉型：工作类卡出现概率+20%
- 温和型：休息类+社交类出现概率+15%
- 放任型：特殊事件卡出现概率+25%

## UI设计

- 横向展开，类杀戮尖塔风格

- 三消提示：当已有2张同类型时显示"再选1张×工作即可三消！"

- 顶部显示"第X月 · 上/中/下旬 · 手牌X张"
- 当前阶段提示清晰
