# 牌阵三消任务系统设计

## 概述
将原有的连击系统（combo）替换为牌阵三消系统，任务选择从列表改为3×3九宫格牌阵。玩家通过匹配3张同类型任务牌获得三消奖励，匹配完成后推进1个月（上/中/下旬）。

## 核心流程
```
每月初 → 生成3×3任务牌阵 → 玩家选3张同类型牌
  → 三消成功 → 3张牌依次执行（上旬→中旬→下旬）
  → 结算三消奖励 → 推进1个月 → 进入下一月
```

## 牌阵规则
- **牌阵**：3×3 九宫格，每月初自动刷新
- **牌类型**（4类）：
  - 📋 **工作**（蓝色边框）：公文处理、会议服务、调研任务
  - 🍵 **社交**（绿色边框）：社交应酬
  - ☕ **休息**（橙色边框）：休息放松
  - 📚 **学习**（紫色边框）：学习充电
- **选择逻辑**：
  - 点击卡牌→高亮选中，同类型累积计数
  - 最多选3张，自动匹配同类型
  - 点击不同类型→重置当前选择
  - 选满3张同类型→自动触发三消

## 刷新机制
- 每月提供1次免费刷新
- 额外刷新消耗50精力
- 若牌阵无可用配对（同类型不足3张），自动刷新并提示

## 三消奖励

| 三消类型 | 奖励 |
|---|---|
| 工作×3 | 声望+10，经验×1.5倍 |
| 社交×3 | 人脉+8 |
| 休息×3 | 精力+20，压力-15 |
| 学习×3 | 经验+30 |

## 时间推进
- 一次三消 = 上旬→中旬→下旬三次执行 = 推进1个月
- 每次执行分别消耗对应任务的精力/压力

## UI变更
- 状态栏：🔥连击显示 → 🎴配对进度显示（"当前配对：工作 2/3"）
- 选择界面：列表选择 → 3×3卡片网格（颜色边框+类型图标）
- 选中动画：发光+放大效果
- 三消成功：庆祝特效
- 无匹配：刷新提示
