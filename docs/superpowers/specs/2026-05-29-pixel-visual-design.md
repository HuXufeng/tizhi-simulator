## 一、视觉主张

> **像素风 × 摸鱼文化 × 幽默轻松**
> 
> 谁说体制内就一定是压抑的？在这里，像素小人们在格子间里摸鱼、偷懒、喝咖啡、聊天——生活气息满满，轻松又治愈。

**色调**: 以暖色、柔和色为主（米白、淡粉、鹅黄、薄荷绿）
**氛围**: 明亮的办公室，窗外永远是好天气（除非你压力爆表）
**幽默元素**: 每个场景藏着小彩蛋，等你发现

## 三、详细设计

### 3.1 PixelSceneCanvas 场景绘制系统

**定位**: 游戏主要视觉背景，全屏Canvas像素画

**场景列表** (8个主要场景 + 4个特殊场景):

| 场景ID | 名称 | 描述 | 关键元素 |
|--------|------|------|----------|
| office | 🏢 办公室 | 温馨格子间 | 电脑、绿植、水杯、日历、摸鱼小人 |
| meeting | 🪑 会议室 | 会议室 | 长桌、椅子、投影仪、白板 |
| cafeteria | 🍱 食堂 | 午餐食堂 | 打饭窗口、餐桌、饭菜热气 |
| corridor | 🚶 走廊 | 办公走廊 | 公告栏、盆景、饮水机 |
| leader_office | 🛋️ 领导办公室 | 宽敞办公室 | 大班桌、书柜、茶具 |
| rest_area | ☕ 茶水间 | 摸鱼圣地 | 咖啡机、零食、微波炉 |
| entrance | 🏛️ 机关门口 | 单位大门 | 门牌、值班室、石狮子 |
| rooftop | 🌤️ 天台 | 摸鱼秘境 | 天空、栏杆、花盆、小凳子 |
| *special_elevator* | *🛗 电梯* | *电梯内* | *按键面板、楼层显示* |
| *special_document* | *📄 文件堆* | *文件山* | *堆积如山的文件夹* |
| *special_afterwork* | *🌆 下班路上* | *下班* | *夕阳、路灯、背影* |
| *special_sickbed* | *🏥 病床* | *加班过度* | *病床、吊瓶（很罕见）* |

**时间段变体** (3种光照):
| 时间段 | 调色板变体 | 氛围描述 |
|--------|------------|----------|
| morning (6-12) | 暖黄+米白 | 阳光斜射，温暖明亮 ☀️ |
| afternoon (12-18) | 正常+柔和 | 明亮舒适，适合摸鱼 😊 |
| evening (18-24) | 暖橙+深蓝 | 窗外夜景，室内灯光温馨 🌙 |

**13色调色板** (暖心复古味):
```
#f5f0e8  米白         #e8ddd0  浅米  
#d4c5b0  中米         #8b7355  棕色
#5c4a3a  深棕         #000000  黑色
#ffffff  白色         #f0a8a8  粉红
#f7d794  鹅黄         #a8d8b9  薄荷
#7ec8e3  天蓝         #e8a87c  橙
#c9b1ff  淡紫
```

**动态小物件**:
| 物件 | 动画 | 放置场景 |
|------|------|----------|
| 电脑屏幕 | 屏保飘动/待机闪烁 | office |
| 风扇 | 旋转 | office |
| 绿植 | 轻微摇摆 | office, corridor, leader_office |
| 咖啡热气 | 蒸汽上升 | cafeteria, rest_area |
| 时钟指针 | 缓慢转动 | office, meeting |
| 饮水机气泡 | 气泡上升 | corridor |
| 窗外小鸟 | 飞过 | all |
| 公告栏纸条 | 随风吹动 | corridor |

**隐藏彩蛋**:
| 触发条件 | 彩蛋内容 |
|----------|----------|
| 连续摸鱼3次 | 茶水间出现一只偷吃零食的像素老鼠 🐭 |
| 精力>90 | 办公室出现彩虹 |
| 成就"摸鱼大师" | 场景中的小人变成摸鱼姿势 |
| 周五下午 | 所有NPC小人开始看表 |
| 连击>5 | 场景出现✨特效 |

**技术规格**:
- 画布尺寸: 320×192 像素 (固定分辨率)
- 渲染方式: 双线性放大至容器宽度的60%
- 帧率: 12 FPS (低功耗)
- 缓存在 offscreenCanvas
- 仅在场景变化时重绘

| mischievous 😏 | 粉红 #f5a0a0 | 😏 | 调皮粉色背景 |

**新增：摸鱼事件专属卡片**:
- "悄悄摸鱼" - 像素猫猫探头动画
- "偷懒被发现" - 被抓包的像素小人
- "摸鱼分享" - 分享零食的温馨画面

**卡片结构**:
```
┌──────────────────────────┐
│  [像素图案装饰]           │  <- 18x18像素图案
│                          │
│  📢 事件标题              │  <- 标题
│  事件描述文字...          │  <- 描述
│                          │
│  ┌──────┐  ┌──────┐    │  <- 决策按钮

│  └──────┘  └──────┘    │
└──────────────────────────┘
```

**动画效果**:
- 入场: 像素方块组合动画 (逐块出现)
- 悬停: 按钮像素化发光边框
- 选择: 确认的像素勾勾动画

### 3.5 PixelUI 像素化UI系统

**定位**: 将游戏UI元素像素化，统一视觉风格

**像素按钮**:
```
┌──────────────────────┐
│  [ 确定 ]            │  <- 像素边框 (3px solid)
│   ██████████████     │  <- 内填充线 (1px)
└──────────────────────┘
```
- 悬停: 像素发光边框
- 点击: 凹陷效果 (像素下移1px)
- 禁用: 灰色 + 像素十字纹

**像素进度条**:
```
精力 ████████░░░░ 80/100
      ↑ 实心像素块 ↑ 空心矩形
```
- 精力: 绿色像素块 (高) → 黄色 (中) → 红色 (低，闪烁)
- 压力: 从空 → 满，颜色渐变绿色→黄色→红色
- 带像素箭头标记

**像素面板**:
```
┌══════════════════════┐  <- 双线像素边框
║ 标题                 ║
║ ─────────────────── ║  <- 像素分隔线
║ 内容                 ║
║                      ║
║ [按钮]  [按钮]       ║
└══════════════════════┘
```

**状态图标化**:
| 状态 | 像素图标 | 颜色 | 动画 |
|------|----------|------|------|
| 精力 | ⚡ | 绿→黄→红 | 低时闪烁 |
| 压力 | 💥 | 绿→橙→红 | 高时抖动 |
| 声望 | 🌟 | 灰→银→金 | 升级时闪光 |
| 人脉 | 🤝 | 蓝 | 增加时握手动画 |
| 金钱 | 💰 | 金 | 增加时金币落下 |

**鼠标悬停反馈**:
- 可交互元素: 像素放大镜效果 (放大120%)
- 提示文字: 像素对话框
- 点击涟漪: 像素方块扩散

## 四、摸鱼气质视觉设计

### 4.1 轻松愉快色调
```
主色:   #f5f0e8 (暖暖米白)    → 背景
辅色:   #a8d8b9 (清新薄荷)    → 积极元素
点缀:   #f7d794 (阳光鹅黄)    → 高亮
警示:   #f0a8a8 (温柔粉红)    → 危险/低值
文字:   #5c4a3a (暖棕)        → 可读性最佳
```

### 4.2 趣味细节设计

**摸鱼成功时**:
- 场景切换为"带薪摸鱼"模式
- NPC眨眼 + 头顶😏
- 窗外小鸟排队飞过
- 背景颜色变暖调

**压力高时**:
- 场景添加灰暗滤镜
- NPC表情变😰
- 有叹气特效
- 时钟变快（焦虑暗示）

**连击高时**:
- 角色做加油动作
- 舞台特效 (聚光灯)
- 场景中撒🎉
- 背景节奏闪烁

**摸鱼被发现时**:
- "被抓包"的音效
- 场景震动一下
- 所有人扭头看你
- 表情全部变成😳

### 4.3 幽默彩蛋

| 条件 | 彩蛋 |
|------|------|
| 连续摸鱼5次成功 | 茶水间多了一只像素猫 🐱 |
| 周五下午摸鱼 | 场景中的NPC都在玩手机 |
| 被批评后摸鱼 | 说明文字变成"化悲痛为摸鱼" |
| 成就"摸鱼宗师"解锁 | 所有场景出现🐟图标 |
| 和领导关系好时摸鱼 | 领导也在一起摸鱼 😂 |
| 精力为0时 | 角色变成躺平的像素小人 |
| 摸鱼失败10次 | 获得成就"非酋摸鱼人" |

### 6.1 与 eventSystem 集成

```javascript
// 触发事件时
eventTriggered(event) {
  // 根据事件类型绘制场景
  pixelCanvas.drawScene(event.scene || 'office', timeSystem.getTimePart());
  // 显示事件卡片
  sceneCard.show(event.category, event);
  // 添加角色反应
  if (event.relatedNarrator) {
    charSprite.show(event.relatedNarrator, 'thinking');
  }
}

// 决策后
decisionMade(choice) {
  // 结果动画
  sceneCard.showResult(choice.outcome);
  // 角色情绪反应
  charSprite.showMood(choice.emotion);
  // 触发放大效应
  effectLayer.trigger(choice.result === 'success' ? 'confetti' : 'sigh');
}
```

## 七、性能考虑

### Canvas 性能优化
- 固定 320×192 像素分辨率，放大渲染
- 12 FPS 帧率锁定
- offscreenCanvas 场景缓存
- 仅动态元素层每帧重绘

### CSS 性能
- 动画使用 transform + opacity (GPU加速)
- CSS contain 属性隔离重绘
- will-change 通知浏览器

### 内存管理
- 每个场景预渲染到 offscreenCanvas
- 动画完成后清除临时元素
- 长特效设置自动超时
